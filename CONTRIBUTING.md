

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone the project** to your own machine.
3.  **Set up the development environment** by following the instructions in the [README.md](README.md).

## Workflow

1.  Create a new branch for your feature or bugfix:
    ```bash
    git checkout -b feature/amazing-new-feature
    # or
    git checkout -b fix/annoying-bug
    ```
2.  Make your changes.
3.  Commit your changes with descriptive messages.
4.  Push your branch to your fork:
    ```bash
    git push origin feature/amazing-new-feature
    ```
5.  Open a **Pull Request** on the original repository.

## 👥 How to Add a Collaborator (For Repository Owners)

If you are the owner of this repository and want to give your friend full access to push code directly, follow these steps:

1.  **Go to your Repository on GitHub.**
    (This should be `https://github.com/NxAdx/Student-Success-Hub`)
2.  Click on the **Settings** tab (usually the rightmost tab in the top menu).
3.  In the left sidebar, click on **Collaborators** (under the "Access" section).
4.  Click the green **Add people** button.
5.  **Search** for your friend's GitHub username or email address.
6.  Select them and click **Add [username] to this repository**.
7.  **Send the invite link**: GitHub will send them an email, but you can also copy the invite link and send it to them directly.
8.  Once they accept, they will have **Write** access and can push code directly to the repository!
